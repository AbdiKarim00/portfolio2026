## Phase 2: Essential Setup (30 minutes)

### Add Your Projects
- [ ] Create `/public/images/` folder
- [ ] Save first project image (1200Ã—800px JPG)
- [ ] Open `lib/portfolio-data.ts`
- [ ] Add first project to projects array
- [ ] Save (Ctrl+S) and verify it appears
- [ ] Add second project
- [ ] Add third project
- [ ] Verify all projects display correctly

**Time:** ~15 minutes for 3 projects  
**Status:** Portfolio with projects

### Set Up CV Download
- [ ] Create `/public/documents/` folder
- [ ] Add your CV PDF file
- [ ] Open `components/portfolio-about.tsx`
- [ ] Find `handleDownloadCV` function (line ~46)
- [ ] Update `cvPath` to your CV location
- [ ] Save (Ctrl+S)
- [ ] Test CV download button works

**Time:** ~5 minutes  
**Status:** CV downloadable

### Personalize Your Bio
- [ ] Open `components/portfolio-about.tsx`
- [ ] Find "Background" section (~line 82)
- [ ] Replace with your professional story

**Status:** Your voice and story

**Status:** Fully functional personalized portfolio

## Troubleshooting Checklist

If something isn't working:

### Images not showing?
- [ ] Image in `/public/images/` folder?
- [ ] Correct file path in data?
- [ ] File name spelled correctly?
- [ ] JPG format used?
- [ ] File size reasonable?

### CV not downloading?
- [ ] PDF in `/public/documents/` folder?
- [ ] Path correct in `portfolio-about.tsx`?
- [ ] File name matches?
- [ ] PDF file is valid?

### Changes not appearing?
- [ ] File saved (Ctrl+S)?
- [ ] Page refreshed?
- [ ] Browser cache cleared?
- [ ] Correct file edited?

### Mobile view broken?
- [ ] Responsive classes added?
- [ ] Viewport meta tag exists?
- [ ] Breakpoints correct?
- [ ] Test on actual device?
